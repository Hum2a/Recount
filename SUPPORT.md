# Support

## How to get help

- **Bug?** Open a [bug report](https://github.com/Hum2a/Recount/issues/new?template=bug_report.yml) with steps to reproduce.
- **Idea?** Use the [feature request](https://github.com/Hum2a/Recount/issues/new?template=feature_request.yml) template.
- **Security?** Do not file a public issue — see [`SECURITY.md`](./SECURITY.md).
